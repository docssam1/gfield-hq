## 문제
`fillComment()`의 코멘트가 세 축 단순 격차(gap = 최대-최소)만 보고 "사고력이 N개월 뒤처져 더 확인 필요"라고만 출력.
- ① **연산이 교과보다 뒤처진 "순서 위반(잘못된 배치)"을 감지 못 함** (재설정 경고 없음)
- ② **교과는 됐는데 사고력만 뒤처진 "좋은 신호"를 "확인 필요"로 잘못 처리**
- 화면의 "사고력 7개월" = 세 축 최대-최소 격차일 뿐, 레벨 카드 수치(4/8/11)와 무관 → 혼란.

## 판정 기준 (shift 단위=개월, 클수록 빠름. shS=교과, shC=연산, shT=사고력)
- ① 연산<교과 역전: `shS - shC >= 2` → 재설정 경고 (최우선)
- ② 사고력만 뒤처짐: `|shS - shC| < 2 && shS - shT >= 2 && shC - shT >= 2` → 좋은 신호
- ③ 그 외: 기존 로직 유지

## 수정 1 — 축 계산 직후 판정 플래그 추가

### 찾기 (정확히 일치, 두 파일 공통)
```
  const ahead=sorted[0], behind=sorted[2];
  const gap=behind[1]-ahead[1];

  const tKey=cur.thinking1?.key||'사고력 과정';
```

### 바꾸기
```
  const ahead=sorted[0], behind=sorted[2];
  const gap=behind[1]-ahead[1];

  // === 균형 순서 진단 (연산 ≥ 교과 ≥ 사고력 이 정상) ===
  // shift: 클수록 빠른 진도. shS=교과, shC=연산, shT=사고력.
  const calcBehindSchool = (shS - shC) >= 2;            // 교과가 연산보다 2개월+ 앞섬 = 연산이 못 받쳐줌 (순서 위반)
  const schoolCalcBalanced = Math.abs(shS - shC) < 2;   // 교과·연산은 균형
  const thinkingOnlyBehind = schoolCalcBalanced && (shS - shT) >= 2 && (shC - shT) >= 2; // 사고력만 뒤처짐 = 좋은 신호

  const tKey=cur.thinking1?.key||'사고력 과정';
```

## 수정 2 — "2. 균형 + 뒤처진 축" 분기 교체

### 찾기 (정확히 일치, 두 파일 공통)
```
  // 2. 균형 + 3. 뒤처진 축
  if(gap<=1){
    c+=`교과·사고력·연산이 비교적 고르게 자리 잡고 있습니다.</p>`;
  } else {
    c+=`다만 <b>${ahead[0]}</b>${josaEunNeun(ahead[0])} 상대적으로 앞서 있는 반면, <b>${behind[0]}</b>${josaEunNeun(behind[0])} 약 ${gap}개월가량 뒤처져 더 확인이 필요합니다.</p>`;
  }
```

### 바꾸기
```
  // 2. 균형 + 3. 뒤처진 축 / 순서 진단
  if(calcBehindSchool){
    // ① 연산<교과 역전 = 잘못된 배치 → 재설정 경고 (최우선)
    c+=`다만 <b style="color:#b91c1c">교과 진도가 연산보다 앞서 있어, 연산이 교과를 충분히 받쳐주지 못하는 상태</b>입니다. 연산이 받쳐주지 않은 채 교과만 나가면 구멍이 생기기 쉽습니다. 지금은 교과 진도를 잠시 멈추고, <b>연산을 교과 수준에 맞춰 다시 설정</b>하는 것이 먼저입니다.</p>`;
  } else if(thinkingOnlyBehind){
    // ② 교과·연산은 잡혔고 사고력만 뒤처짐 = 좋은 신호
    c+=`교과와 연산이 잘 자리 잡았고 <b>사고력이 그 뒤를 따라오는 좋은 흐름</b>입니다. 교과에서 단위 쓰기·조건 보기가 되면 지문 이해와 사고력은 자연스럽게 따라붙습니다. 교과·연산에 쓰던 시간의 일부를 <b>사고력에 할애해 속도를 붙여도 좋은</b> 단계입니다.</p>`;
  } else if(gap<=1){
    c+=`교과·사고력·연산이 비교적 고르게 자리 잡고 있습니다.</p>`;
  } else {
    c+=`다만 <b>${ahead[0]}</b>${josaEunNeun(ahead[0])} 상대적으로 앞서 있는 반면, <b>${behind[0]}</b>${josaEunNeun(behind[0])} 약 ${gap}개월가량 뒤처져 더 확인이 필요합니다.</p>`;
  }
```
